# İngilizce - Türkçe Elektronik Terimler Sözlüğü

| English | Türkçe | Not |
|---|---|---|
| Voltage | Gerilim / Voltaj | İki nokta arasındaki potansiyel fark |
| Current | Akım | Bir elemanın içinden akan yük |
| Resistance | Direnç | V/I ilişkisini belirleyen büyüklük |
| Power | Güç | P = V × I |
| Ground | Toprak / Referans | Devrenin referans noktası |
| Chassis | Şase | Mekanik/koruyucu yapı referansı |
| Voltage divider | Gerilim bölücü | Giriş geriliminin belirli oranını üretir |
| Thevenin equivalent | Thevenin eşdeğeri | Kaynak + seri direnç modeli |
| Capacitor | Kapasitör | Elektrik alanında enerji depolar |
| Inductor | Endüktör | Manyetik alanda enerji depolar |
| Diode | Diyot | Akımı ağırlıklı olarak tek yönde iletir |
| Rectifier | Doğrultucu | AC/DC dönüşümünde kullanılır |
| Impedance | Empedans | AC devrede genelleştirilmiş direnç |
| Reactance | Reaktans | Kapasitif/endüktif frekans bağımlı etki |
| Op-amp | İşlemsel yükselteç | Geri besleme ile kullanılan yüksek kazançlı yükselteç |
| Comparator | Karşılaştırıcı | İki gerilimi karşılaştırıp dijital benzeri çıkış üretir |
| Pull-up | Yukarı çeken direnç | Hattı lojik 1'e zorlar |
| Pull-down | Aşağı çeken direnç | Hattı lojik 0'a zorlar |
| Termination | Sonlandırma | İletim hattı yansımalarını azaltır |
| Bypass capacitor | Bypass kapasitörü | Besleme gürültüsünü bastırır |
| Coupling capacitor | Kuplaj kapasitörü | DC'yi bloke edip AC sinyali geçirir |
