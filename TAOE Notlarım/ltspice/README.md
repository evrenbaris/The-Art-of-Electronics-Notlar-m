# LTspice Örnekleri

Bu klasörde basit `.cir` netlistleri bulunur. Amaç kitap devrelerini kopyalamak değil, kavramları kendi simülasyonlarınla gözlemlemektir.

## Çalıştırma

LTspice → File → Open → `.cir` dosyasını seç → Run.

## Örnekler

- `01-voltage-divider.cir`
- `02-rc-lowpass.cir`
- `03-rc-highpass.cir`
- `04-diode-rectifier.cir`
- `05-bjt-switch.cir`
- `06-mosfet-low-side.cir`
- `07-opamp-inverting-ideal.cir`
- `08-buck-concept.cir`
