# İngilizce - Türkçe Elektronik Sözlüğü

| English | Türkçe | Kısa açıklama |
|---|---|---|
| Voltage | Gerilim / voltaj | İki nokta arasındaki potansiyel fark |
| Current | Akım | Yük akış hızı |
| Resistance | Direnç | Gerilim-akım oranı |
| Conductance | İletkenlik | Direncin tersi |
| Power | Güç | Birim zamanda enerji |
| Ground | Toprak / referans | Devrede seçilen referans düğüm |
| Chassis | Şase | Mekanik gövde / koruyucu iletken yapı |
| Return | Dönüş hattı | Akımın kaynağa geri döndüğü yol |
| Bypass capacitor | Bypass kapasitörü | Besleme gürültüsünü lokal bastırır |
| Decoupling | Ayrıştırma / besleme bypass | IC beslemesini anlık akım için destekleme |
| Thevenin equivalent | Thevenin eşdeğeri | Kaynak + seri direnç eşdeğeri |
| Norton equivalent | Norton eşdeğeri | Akım kaynağı + paralel direnç eşdeğeri |
| Impedance | Empedans | AC devrede gerilim/akım oranı |
| Reactance | Reaktans | Kapasitif/endüktif empedansın sanal bileşeni |
| Common-mode | Ortak mod | İki hattın birlikte hareket eden bileşeni |
| Differential | Diferansiyel | İki hat arasındaki fark bileşeni |
| Termination | Sonlandırma | Hat empedansını eşleştiren direnç/ağ |
| Hysteresis | Histerezis | Eşiklerde kararsızlığı azaltan pozitif geri besleme |
| Offset | Ofset | Sıfır girişte kalan hata |
| Bias current | Bias akımı | Amplifikatör girişine akan küçük akım |
| Slew rate | Yükselme/eğim hızı sınırı | Op-amp çıkışının maksimum değişim hızı |
| Settling time | Yerleşme süresi | Çıkışın hata bandına girme süresi |
| Noise | Gürültü | İstenmeyen rastgele veya deterministik sinyal |
| Shield | Ekran/kalkan | Elektrik/manyetik kuplajı azaltan iletken yapı |
| Ground loop | Toprak döngüsü | İstenmeyen dönüş akımı döngüsü |
| Pull-up | Yukarı çeken direnç | Hattı varsayılan HIGH yapar |
| Pull-down | Aşağı çeken direnç | Hattı varsayılan LOW yapar |
| Open drain | Açık drain | Sadece LOW süren çıkış tipi |
| High-side switch | Üst taraf anahtar | Yükün besleme tarafını anahtarlayan devre |
| Low-side switch | Alt taraf anahtar | Yükün dönüş tarafını anahtarlayan devre |
| Inrush current | İlk kalkış akımı | Açılışta kondansatör/yük nedeniyle oluşan yüksek akım |
| Derating | Pay bırakma | Komponenti limitinin altında kullanma |
