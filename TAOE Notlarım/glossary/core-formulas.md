# Temel Formül Kartları

## DC

```text
V = I·R
P = V·I = I²R = V²/R
```

## RC

```text
τ = R·C
Vcharge(t) = Vf(1 - e^(-t/RC))
Vdischarge(t) = V0 e^(-t/RC)
fc = 1/(2πRC)
```

## LC

```text
Xc = 1/(2πfC)
Xl = 2πfL
f0 = 1/(2π√(LC))
```

## Op-amp

```text
Inverting gain = -Rf/Rin
Non-inverting gain = 1 + Rf/Rg
Slew requirement ≈ 2πfVp
```

## ADC

```text
LSB = Full Scale / 2^N
Quantization RMS ≈ LSB/√12
```

## Termal

```text
Tj = Ta + P·θJA
```
