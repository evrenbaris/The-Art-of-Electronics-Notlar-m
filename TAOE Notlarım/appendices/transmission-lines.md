# Transmission Line Notları

## Ne zaman hat gibi davranır?

Sinyal yükselme süresi, kablo yayılım gecikmesine yakınsa bağlantı artık basit tel değildir.

## Temel kavramlar

- Characteristic impedance
- Far-end termination
- Source termination
- Reflection
- Return path

## Aviyonik bağlantı

Harness uzunluğu, connector pin dizilimi, shield termination ve ground reference veri linki güvenilirliğini doğrudan etkiler.
