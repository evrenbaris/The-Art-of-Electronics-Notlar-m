# Telif Güvenli Çalışma Akışı

## Güvenli yöntem

1. Kitaptaki konuyu oku.
2. Kitabı kapat.
3. Konuyu kendi kelimelerinle anlat.
4. Kendi devre örneğini çiz.
5. Kendi simülasyonunu kur.
6. Kendi hatalarını ve çıkarımlarını yaz.

## Kaçınılacak davranış

- Paragraf paragraf çeviri yapmak
- Kitap figürlerini yeniden çizip aynı şekilde yayınlamak
- Kitap tablolarını markdown tablosu olarak aktarmak
- PDF'i repoya eklemek

## İyi örnek

Yanlış:

```text
Chapter 1'in birebir Türkçe çevirisi
```

Doğru:

```text
Gerilim bölücü kavramını ADC batarya ölçüm örneğiyle anlatan özgün not
```
