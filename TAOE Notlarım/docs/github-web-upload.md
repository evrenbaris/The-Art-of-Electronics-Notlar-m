# GitHub Web Arayüzü ile Yükleme

1. Bu ZIP dosyasını bilgisayarına indir.
2. ZIP'i çıkar.
3. GitHub'da boş repo oluştur.
4. Repoda **Add file → Upload files** seç.
5. ZIP dosyasını değil, ZIP'ten çıkan klasörün içindeki dosya/klasörleri sürükle.
6. Commit mesajı yaz:

```text
Initial commit: add Turkish electronics design notes
```

7. **Commit changes** butonuna bas.

> Not: GitHub boş klasörleri göstermez. Bir klasör görünmüyorsa içine README.md veya .gitkeep ekle.
